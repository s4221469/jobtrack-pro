# JobTrack Pro

Job Application Tracking Platform built with FastAPI + React + PostgreSQL.

## How to Run

```bash
docker-compose up --build
```

## Ports

| Service    | Port |
|------------|------|
| Frontend   | 3000 |
| Backend    | 8000 |
| PostgreSQL | 5432 |

## Database Credentials

- **User:** jobtrack

- **Database:** jobtrackdb

## Usage

1. Open `http://localhost:3000`
2. Register a new account
3. Login
4. Add companies, then add job applications
5. Track status and view analytics on the Dashboard
